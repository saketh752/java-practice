# Student Management System

## Description

A simple Java console-based application developed as part of my Java learning journey.

## Features

* Add Student Details
* Display Students
* Search Student by Name
* Find Student with Highest Marks
* Calculate Average Marks

## Concepts Used

* Arrays
* Strings
* Loops
* Conditional Statements
* Linear Search

## Technologies

* Java
* IntelliJ IDEA

## Learning Outcome

This project helped me understand how to combine arrays, strings, loops, and searching techniques to build a simple real-world application.
